# GrantAI Europe

Aplicație web pentru:
- profilul organizației;
- proiecte europene;
- căutarea apelurilor;
- evaluarea eligibilității;
- pregătirea aplicațiilor;
- generarea structurii proiectului.

## Publicare pe Streamlit Cloud

1. Încarcă toate fișierele din acest folder în repository-ul GitHub.
2. Intră pe Streamlit Community Cloud.
3. Selectează repository-ul `GrantAI-Europe`.
4. Main file path: `app.py`.
5. Apasă Deploy.

Important: în GitHub trebuie încărcate fișierele din interiorul folderului, nu arhiva ZIP.
